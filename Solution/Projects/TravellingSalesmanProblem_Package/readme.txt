Project requirements:
1. C++17
2. CMake version 3.8 or higher
3. Built with Visual Studio 16 2019

To execute:
1. Follow the instructions given in CMake_readme.txt to create the executable.
2. Once the executable is created, in the commandline,please type: /path/to/executable /path/to/project and then run.
3. Further instructions will be available with the help menu that shows up.
